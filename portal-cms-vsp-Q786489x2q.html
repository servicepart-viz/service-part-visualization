SOP / Panduan Penggunaan CMS

1. Konfigurasi Awal (Login)

Agar bisa menyimpan data, Anda membutuhkan Personal Access Token (PAT) dari GitHub.

  • Buka konfigurasi GitHub: Settings > Developer settings > Personal access tokens > Tokens (classic).
  • Generate new token, beri nama, dan centang opsi scopes 'repo'.
  • Copy token tersebut (berawalan ghp_) dan paste ke form login Admin Panel ini.

2. Menambah & Mengelola Menu

Konfigurasi website terbagi dalam Menu dan Tabs.

  • Klik + Add Menu untuk membuat kategori navigasi baru.
  • Menu ID harus unik dan tanpa spasi (misal: WAREHOUSE).
  • Menu Label adalah teks yang memanjang dan akan tampil di navbar atas.

3. Mendapatkan Report ID Power BI

  • Buka report di Power BI Service.
  • Pilih menu File > Embed report > Website or portal.
  • Perhatikan URL pada popup yang muncul. Salin bagian ID-nya.
    Contoh: ...reportId=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx&ctid...
  • Paste ID tersebut ke dalam kolom Power BI Report ID pada Form.

4. Menyimpan Perubahan

Setelah selesai mengubah Menu/Tab, klik Save & Publish to GitHub di pojok kanan bawah. Sistem akan otomatis melakukan commit ke repository GitHub Anda. Mohon tunggu beberapa detik hingga muncul notifikasi keberhasilan.
